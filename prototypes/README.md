# Prototypes & Design References

Esta pasta serve para armazenar protótipos de design, arquivos exportados de ferramentas de UI (como Nano Banana, Figma, etc.) e referências visuais que não fazem parte do código-fonte Angular principal.

## Como usar

1.  Cole aqui pastas ou arquivos gerados por ferramentas externas (ex: projetos React/Vite exportados).
2.  Estes arquivos **não** serão processados pelo build do Angular.
3.  Use estes arquivos como referência para copiar/adaptar HTML (Tailwind) e lógica para os componentes Angular em `src/app`.
